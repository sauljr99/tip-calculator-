Tip Calculator

A small desktop app I built that calculates the tip and total bill
for a meal. You enter the bill amount and number of people, pick a
tip percentage using the radio buttons (10%, 15%, 20%, or 25%),
and it splits everything evenly and shows how much each person owes.

Language: Python
Libraries: Tkinter (for the GUI window, buttons, and input boxes),
           tkinter messagebox (pops up an error if you type something invalid)

Notes:
- Run the .py file and the window will open.
- The result shows the tip amount, total bill, and split per person
  all on one line.
- If you type letters instead of numbers it will show an error box.
